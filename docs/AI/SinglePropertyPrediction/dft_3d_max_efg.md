# Model for max_efg
<!--benchmark_description--> - Description: This is a benchmark to evaluate how accurately an AI model can predict the maximum value of electric field gradient using the JARVIS-DFT (dft_3d) dataset. The dataset contains different types of chemical formula and atomic structures. Here we use mean absolute error (MAE) to compare models with respect to DFT (meta-GGA TBmBJ) accuracy.
